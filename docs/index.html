ScriptuccinoJS is a simple set of JavaScript function and utility modules for easily performing some common tasks.

Usage

The following examples assume that the ScriptuccinoJS modules and library script are being served from a directory named /scriptuccino/.

ES modules

Import the modules you need into your javascript:

Use static import:

import whenPageLoaded from '/scriptuccino/modules/events/page/whenPageLoaded.js';

const myFoo = () => alert('foo');

whenPageLoaded(myFoo);

Using dynamic import:

const myFoo = () => alert('foo');

import('/scriptuccino/modules/events/page/whenPageLoaded.js').then(
    (module) => module.whenPageLoaded(myFoo)
);

Using dynamic import with await:

const myFoo = () => alert('foo');

const module = await import('/scriptuccino/modules/events/page/whenPageLoaded.js');

module.whenPageLoaded(myFoo);

ES5 Library

Add the library to your webpage, preferably at the bottom of the body - but above your own scripts (external or inline):

…
<body>
<h1>Foo</h1>
<p>Lorem ipsum dolor sit amet...</p>

<script src="/scriptuccino/scriptuccino.js"></script>
<script src="/your.own.script.js"></script>
</body>
…

In your own scripts, access the ScriptuccinoJS modules and use the functions you need:

const myFoo = () => alert('foo');

SCRIPTUCCINO.events.whenPageLoaded(myFoo);

API

Modules

ScriptuccinoJS is organised into the following modules, each containing a number of utilities. If using ScriptuccinoJS as ES2015/ES modules you can import them from any of the files they are documented as being exported from below. If using the ES5 library, you can access them from the `SCRIPTUCCINO` global/namespace object as described.

utils

Tools for doing common things with a little more ease.

elementComesIntoViewport( elements: Element | Nodelist, fn: Function[, options: Object] )

uses: elementScrolledAboveFold (as fallback)

WARNING: This utility uses experimental web features (Intersection Observer) which is not yet finalised and could change. IT MAY BREAK/THROW ERRORS if the user agent uses a different implementation.

It fallsback to elementScrolledAboveFold functionality if Intersection Observer is not supported at all.

Call the supplied function as soon as the specified element is scrolled into the viewport using the IntersectionObserver API.

The callback function fn receives 2 arguments, the first is the element that has entered the viewport and the second is the IntersectionObserver instance.

The options argument is a plain object to customise the observer that is created. Please see IntersectionObserver API documents for details on what properties this can take.

There is only one IntersectionObserver instance created for all nodes that are passed to this utility. The element will continue to trigger intersections unless you unobserve it in the supplied function.

fx

Programmatically apply CSS transitions. Great for throw-away transitions, prototyping or quickly applying complex effects without having to figure out the CSS.

applyTransition( element: HTMLElement, settings: Object|Array, styles: Object [, completeFn: Function] )

Define and apply CSS transition to an HTML element, and optionally perform a callback function when the transition is complete.

The optional settings argument takes either a plain object or an array of objects which determine the specification of the transition/s.

The properties and types of this object are:

{
    property: String,
    duration: Number,
    curve: String,
    delay: Number,
}

The number values represent milliseconds. The property property is the CSS property to transition, the curve is the timing function to use, which must be provided as a string.

exported as default by:

ES5 library

SCRIPTUCCINO.fx.applyTransition fadeIn( element: HTMLElement [, completeFn: Function, settings: Object] )

uses: applyTransition

Apply a fade-in transition to an HTML element. Optionally perform a callback when complete and define optional settings for the transition.

Default duration is 500ms. Default timing function is ease. The settings object is the same as for applyTransition.
